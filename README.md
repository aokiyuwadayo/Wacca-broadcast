# YUWA Broadcast（仮称）

> サークルの告知を「ふんわりメモ」から、各プラットフォーム向けに最適化して生成・配信するツール。
> 起業部 部長（青木）の告知業務の負担を減らすために作る。将来は他サークルにも開放できる形で設計する。

- **対象**: まずは青木（起業部 部長）1人。将来は他サークルの運営にも。
- **状態**: 設計確定・実地検証済（2026-05-29）。実装はこれから（Phase A から）。
- **設計の全体像**: [`docs/design-doc.md`](docs/design-doc.md)

## なぜ作るか（解決する痛み）

毎週の活動・イベント情報を LINE グループ / Teams（将来 Discord / Notion）に共有する作業が重い：

1. 投稿先が多い（二重・三重投稿）
2. 見やすい文章を作るのが面倒
3. 先延ばし・投稿忘れ（活動前日になりがち）

1回 30分〜1時間 × 毎週 ＝ 年間 25〜50時間の負担。

## コンセプト（ひとことで）

> **頭の中のふんわりメモを投げると、AI が相談しながら整え、各 SNS 向けに書き分けて、配信（or コピペ）まで地続きにする。忘れないようリマインドもする。**

詳細は [設計ドキュメント](docs/design-doc.md) を参照。
